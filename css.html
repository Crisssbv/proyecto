CSS

CSS se encarga del diseño visual: colores, tipografías, espaciado, layout y responsive.

Ver comparación Volver al inicio

CSS (Cascading Style Sheets) define la presentación del HTML. Permite aplicar estilos a elementos y organizar el diseño (layout) usando técnicas modernas como flexbox y grid, logrando diseños responsivos.

  • Separa contenido (HTML) de presentación (estilos).
  • Reutiliza estilos en múltiples páginas.
  • Facilita el diseño responsive para móviles y tablets.
  • Mejora la consistencia visual del sitio.

  1. Diseño de menús, botones y formularios.
  2. Layouts con Flexbox y CSS Grid.
  3. Animaciones y transiciones (hover).
  4. Adaptación a múltiples tamaños de pantalla.

Ejemplo rápido

Aplicar tipografía y color:

body {
  font-family: 'Montserrat', sans-serif;
  color: #222;
}
